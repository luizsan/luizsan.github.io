Touhou

One of my first projects in Unity, I tried making a Touhou fangame, which is a bullet hell shooter. All of the graphics were redone very similar to the original but in a much higher resolution, so the game will still look pretty in 1080p and plays almost exactly the same.

Different characters have different shot types, with varying degrees of power and versatility. Collisions are disabled for this demo, so you can graze the bullets but cannot get hit.

Yes, you are supposed to dodge all of it.

Valore Stories

A Final Fantasy Tactics clone. I really love this game, so I took the overall idea and tried to improve it, bringing it to modern standards.

Unfortunately, the mechanics are way too much complicated and I thought it wouldn't be a good to keep investing time in that formula, so the game got reworked into something else.

As shown in the video, you can battle against an AI - and it is pretty hard. Each of the units can have different stats, different skills with different effects, different equipment, and all of it affect how they are played.

AccelBeat

A prototype rhythm game specialized for dance games like Dance Dance Revolution and Pump It Up, inspired by StepMania.

It can read StepMania files from folders and play them. It also implements features like an innovative pointer-controlled interface with tiles and sort modes for Select Music screen.

2017-2018

LS Editor

A song editor for AccelBeat, 100% compatible with StepMania 5. In the video, I pick a song to edit and play, and the same song is shown being played on another software without any inaccuracies.

2017-2018

Versus Tactics

The current game I'm working on. It's a 1v1 strategy game and the objective is to defeat the enemy troops using yours.

You can select an unit every turn to move and act with. Actions can put units on cooldown, so you need to plan them extremely well to emerge victorious. Actions from both players are blind-picked so reading and predicting the enemy actions is also necessary, making it an unique and exciting gameplay.

The video showcases P2P network play between two instances of the game, one being the host and the other connecting as a client.
